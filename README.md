一、实验目的
    ①理解相似变换、仿射变换、透视变换的数学原理和几何特性
    ②观察并总结三类变换对直线、平行线、垂直线、圆等几何性质的影响
    ③掌握使用OpenCV进行透视畸变校正的方法

二、实验原理
2.1 相似变换
    变换矩阵： 旋转 + 平移 + 均匀缩放
    自由度： 4个
    特点： 保持形状相似，角度不变

2.2 仿射变换
    变换矩阵： 线性变换 + 平移
    自由度： 6个
    特点： 平行线保持平行，但角度可改变

2.3 透视变换
    变换矩阵： 3×3投影矩阵
    自由度： 8个
    特点： 产生近大远小效果，平行线可能相交于消失点

三、实验步骤
步骤1：生成测试图像
    创建包含以下元素的600×600测试图像：
    矩形（红色边框）、圆形（蓝色边框）、一组平行线（绿色）、一组垂直线（紫色）

步骤2：施加三种变换
    相似变换： 旋转20° + 缩放0.85倍 + 平移
    仿射变换： 三角形顶点映射实现错切效果
    透视变换： 四边形顶点映射实现透视效果

步骤3：数值验证几何性质
    计算变换前后平行线的斜率差，计算变换前后垂直线的夹角变化，观察圆的形状变化

步骤4：透视畸变校正
    拍摄一张A4纸照片（存在透视畸变），手动点击四个角点 
    使用透视变换将文档校正为500×700的正面视图

四、实验结果
4.1 三种变换效果对比
    三种变换对几何性质的保持能力从强到弱依次为：相似变换 > 仿射变换 > 透视变换。相似变换保留的几何性质最多（直线、平行、垂直、圆），仿射变换次之（保留直线和平行），透视变换保留的性质最少（仅保留直线）。在实际应用中，应根据具体需求选择合适的变换类型。

五、结果分析与讨论
5.1 各变换特性总结
相似变换：
    最"温和"的变换，保持所有几何性质
    圆仍然是圆，直线仍然是直线，角度关系完全不变
    实际应用：图像旋转、缩放、平移
仿射变换：
    保持直线性和平行性
    但会改变角度（直角可能变成锐角或钝角）
    圆会被拉伸成椭圆
    实际应用：图像错切、倾斜校正
透视变换：
    只保持直线性（共线性）
    平行线会相交于消失点（模拟人眼视觉效果）
    圆变成椭圆或复杂的二次曲线
    实际应用：文档校正、3D投影

5.2 透视校正效果评价
本实验采用四点法进行透视校正：
    手动点击文档四个角点是关键步骤，点击精度直接影响校正质量。校正后文字无明显变形，表格线条平直。字符可读性良好，满足文档数字化需求

六、实验结论
    相似变换保持所有几何性质（直线、平行、垂直、圆），是最简单的刚体变换
    仿射变换保持平行性但丧失垂直性和圆的形状
    透视变换只保持共线性，不保持平行和垂直关系
    通过透视变换可以成功校正文档图像的畸变，获得正面视图
